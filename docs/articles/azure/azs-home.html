    UKCloud for Microsoft Azure

    UKCloud for Microsoft Azure harnesses the innovation of Microsoft’s Azure platform delivered from within our award-winning government-grade multi-cloud environment – addressing the service, connectivity and diversity requirements of UK public sector organisations and their industry partners.

    • Manage groups and permissions

    • Retrieve your subscription quotas (using PowerShell)

    • Configure PowerShell environment

    • Use the Azure Stack API

    • Create a Service Principle Name (SPN) (using the Portal)

    • Create a Service Principle Name (SPN) (using PowerShell)

    • Deploy an SQL template (using PowerShell)

    • Link Azure Storage Explorer to Azure Stack
